Typographic Scale & Basic Styles

Standard Styles


Too many type sizes and styles at once can wreck any layout. A typographic scale is a limited set of type sizes that work well together, along with the layout grid. The basic set of styles are based on a typographic scale of 12, 14, 16, 20, 34 and 36.

These sizes and styles balance content density and reading comfort under typical usage conditions.
